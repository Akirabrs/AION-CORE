# AION-CORE: Physics-Aware Adaptive Control Kernel

![Status](https://img.shields.io/badge/Status-TRL%204%20(Lab%20Validated)-success)
![Platform](https://img.shields.io/badge/Platform-FPGA%20%2F%20Real--Time-red)

**AION-CORE** is the unified Control Kernel for Tokamak plasmas, merging **AEGIS** (Safety) and **NPE-PSQ** (Metrics).

## Structure
* `src/hardware`: Verilog RTL (Reflex Core).
* `experiments`: Python Industrial Simulator.
* `docs/academic`: Full Paper & Methodology.
